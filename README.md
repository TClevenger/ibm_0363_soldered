# IBM_0363_soldered

An alternative to the unobtainium/exhorbitantly expensive IBM 0363 (45H2433) port box for the Integrated Netfinity PC on the AS/400 Model 150 and probably others.  This breaks out the PS/2 keyboard and mouse ports from the MDR-40 mini-delta connector on the back of the Netfinity card set.  

This board is tested to fit CUI MD-60S Mini-DIN ports and uses any four-pair cable to connect to a 3M 10140-3000PE "solder cup" style connector.
